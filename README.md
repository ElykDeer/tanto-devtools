# Dev Tools for Tantō
Author: **ElykDeer**

## Description:
Instruction Slices, IL Hierarchy Slices, Dominance Slices, and other tools for devs. Requires [Tantō](https://github.com/Vector35/tanto).

## Minimum Version

This plugin requires the following minimum version of Binary Ninja:

* 4.0.5336
